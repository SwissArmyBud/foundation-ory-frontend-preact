# Foundation-Ory Frontend Template, in Preact/Vite

## Overview

This frontend template is designed as an OAuth client template component for the Foundation-Ory system. It serves as a reference implementation showing how frontend applications can integrate with the Foundation-Ory identity and access management stack.

## Features

- Reference implementation for Foundation-Ory integration
- Demonstrates authentication flows with Ory Hydra as the OAuth provider
- Development and production build scripts

## Getting Started

- `npm run dev` - Starts a dev server at http://localhost:5173/
- `npm run build` - Builds for production, emitting to `dist/`
- `npm run preview` - Starts a server at http://localhost:4173/ to test production build locally
